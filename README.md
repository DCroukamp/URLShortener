# URL Shortener Web Application

A full-stack URL shortening web application built using Node.js, Express.js, PostgreSQL and Redis to generate shortened hyperlinks.

## Features
- Generate shortened URLs from long links
- Redirect users to original URLs using unique short codes
- Improve redirect performance with Redis caching
- Prevent abuse using request rate limiting
- RESTful API architecture for URL creation and retrieval
- Persistent data storage using PostgreSQL

## Technology Stack
- Backend Framework: Express.js (Node.js)
- Database: PostgreSQL
- Caching: Redis
- Rate Limiting: express-rate-limit
- Frontend/UI: Pug Templates, CSS
